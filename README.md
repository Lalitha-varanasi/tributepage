## Tribute Page – Muniba Mazari

This project is a simple and elegant tribute webpage dedicated to Muniba Mazari, also known as the Iron Lady of Pakistan. It highlights her inspiring journey, resilience, and achievements through a visually appealing layout.

## Project Overview

The tribute page is built using basic web technologies and focuses on:
Clean and minimal design
Centered content layout
Styled image with borders and shadows
Informative tribute text section

## Preview

The page displays:
A heading with the name Muniba Mazari
A featured image styled with rounded corners and shadow
A short tribute paragraph describing her life and achievements

## Technologies Used
** HTML5 ** – Structure of the webpage
** CSS3 ** – Styling and layout design

> Download or clone the repository

> Open index.html in any web browser
> git clone <your-repo-link>

> start index.html

## Features
